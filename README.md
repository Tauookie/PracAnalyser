## prac.py - класс Prac
При инициализации принимает два аргумента
- *file* - путь к .txt файлу с данными измерений
- *debug* - нужно ли выводить информацию о промежуточных действиях класса. *True/False*
#### Методы
- *get_variables()* - возвращает список всех доступных переменных.
- *set_axis(axis: 'x'/'y', expr: str)* - задаёт значение, откладываемое по выбранной axis оси. В качестве значения expr может быть как и просто одна из доступных переменных, так и выражение из нескольких.
- *plot_chart()* - создаёт график зависимости в соответствии с осями, указанными в set_axis(). Если оси не заданы ничего не делает.
- *approximate(plot: Bool, start: float, end: float)* - проводит аппроксимирующую прямую и возвращает два значения: коэффициент наклона и свободный член. При *plot=True* также создаст график, наглядно демонстрирующий аппроксимацию. *start* и *end* задают границы аппроксимации.
- *show()* - визуализирует все графики, созданные ранее на отдельных окнах. Блокирует дальнейшее выполнение программы, пока окна не будут закрыты.
